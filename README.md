# 闺蜜情话动画网页

这是一个美丽的网页动画，展示了飘动的彩色文字和彩球，传达闺蜜情谊。

## 在线访问

访问地址：https://你的用户名.github.io/manpingzimu/

## 本地运行

1. 直接双击打开 `index.html`
2. 或在浏览器中打开该文件

## 部署到GitHub Pages

1. 创建GitHub账号并登录
2. 创建新仓库，命名为 `manpingzimu`
3. 上传 `index.html` 文件到仓库
4. 进入仓库设置 → Pages
5. 选择 main 分支作为源
6. 等待部署完成（1-2分钟）
7. 分享链接：https://你的用户名.github.io/manpingzimu/

## 功能

- 彩色彩球不断下落
- 闺蜜情话文字飘动
- 自动适配手机和电脑屏幕
- 24小时可访问
